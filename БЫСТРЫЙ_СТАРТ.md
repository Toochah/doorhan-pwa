# 🚀 БЫСТРЫЙ СТАРТ - Доступ из интернета через GitHub Pages

## ⏱️ Время настройки: 10 минут

---

## 1️⃣ Google Apps Script (5 мин)

1. Откройте: https://script.google.com/
2. "Новый проект"
3. Вставьте код из `pwa/apps-script.js`
4. Нажмите 💾 Сохранить
5. "Развернуть" → "Новое развертывание"
6. Тип: **Веб-приложение**
7. Доступ: **Все** (Anyone)
8. "Развернуть" → Скопируйте URL

---

## 2️⃣ Настройте PWA (1 мин)

1. Откройте `pwa/js/app.js`
2. Строка 362: вставьте URL от Apps Script
3. Сохраните

---

## 3️⃣ Деплой на GitHub (3 мин)

```bash
cd c:\proj\doorhan\pwa
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/ВАШ_USERNAME/doorhan-inspection.git
git branch -M main
git push -u origin main
```

Или запустите: `deploy.bat`

---

## 4️⃣ GitHub Pages (1 мин)

1. Откройте репозиторий на GitHub
2. Settings → Pages
3. Branch: main → Save

---
